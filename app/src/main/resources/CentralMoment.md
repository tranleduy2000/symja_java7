## CentralMoment

``` 
CentralMoment(list, r)
``` 

> gives the the `r`th central moment (i.e. the `r`th moment about the mean) of `list`.
  
See:  
* [Wikipedia - Central moment](https://en.wikipedia.org/wiki/Central_moment)
 

### Examples
``` 
>> CentralMoment({1.1, 1.2, 1.4, 2.1, 2.4}, 4)
0.10085
```
