## GreaterEqual

```
GreaterEqual(x, y) 

x >= y
```

> yields `True` if `x` is known to be greater than or equal to `y`.

```
lhs >= rhs
```
> represents the inequality `lhs >= rhs`.
 
	
### Examples
 
```
>> x>=x
True

>> {GreaterEqual(), GreaterEqual(x), GreaterEqual(1)}
{True, True, True}
```
