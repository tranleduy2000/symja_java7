## PrimeOmega

```
PrimeOmega(n)
```

> returns the sum of the exponents of the prime factorization of `n`. 
 
### Examples  

```
>> PrimeOmega(990)
{{2,1},{3,2},{5,1},{11,1}}
```

```
>> PrimeOmega(341550071728321)
{{10670053,1},{32010157,1}}
```   

```
>> PrimeOmega(2010)
{{2, 1}, {3, 1}, {5, 1}, {67, 1}}
```
