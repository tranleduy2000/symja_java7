## ArcSech

```
ArcSech(z)
```

> returns the inverse hyperbolic secant of `z`.

See:
* [Wikipedia: Inverse hyperbolic function](https://en.wikipedia.org/wiki/Inverse_hyperbolic_function)

### Examples
``` 
>> ArcSech(0)    
Infinity  
  
>> ArcSech(1)    
0

>> ArcSech(0.5)    
1.3169578969248166
```
