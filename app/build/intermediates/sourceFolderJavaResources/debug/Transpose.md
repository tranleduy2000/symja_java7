## Transpose

``` 
Transpose(m)
``` 

> transposes rows and columns in the matrix `m`.

See:
* [Wikipedia - Transpose](https://en.wikipedia.org/wiki/Transpose)

### Examples

``` 
>> Transpose({{1, 2, 3}, {4, 5, 6}})
{{1, 4}, {2, 5}, {3, 6}}
>> MatrixForm(%)
1   4
2   5
3   6
 
>> Transpose(x)
Transpose(x)
```
