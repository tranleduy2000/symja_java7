## ArcTanh

```
ArcTanh(z)
```

> returns the inverse hyperbolic tangent of `z`.

See:
* [Wikipedia: Inverse hyperbolic function](https://en.wikipedia.org/wiki/Inverse_hyperbolic_function)

### Examples
``` 
>> ArcTanh(0)    
0  

>> ArcTanh(1)    
Infinity    
  
>> ArcTanh(.5 + 2*I)
0.09641562020299621+I*1.1265564408348223  

>> ArcTanh(2+I)    
ArcTanh(2+I)
```
