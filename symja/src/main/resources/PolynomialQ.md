## PolynomialQ
 
``` 
PolynomialQ(p, x)
``` 

> return `True` if `p` is a polynomial for the variable `x`. Return `False` in all other cases.
 
``` 
PolynomialQ(p, {x, y, ...})
``` 

> return `True` if `p` is a polynomial for the variables `x, y, ...` defined in the list. Return `False` in all other cases.
