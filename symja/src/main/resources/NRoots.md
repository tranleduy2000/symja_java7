## NRoots

```
NRoots(poly)
```

> gives the numerical roots of polynomial `poly`.

### Examples 
```
>> NRoots(x^3-4*x^2+x+6)
{2.9999999999999996,-1.0000000000000002,1.9999999999999998}
```
