## Symbol
```
Symbol
```
> is the head of symbols.

### Examples
```
>> Head(x)
Symbol
```

You can use `Symbol` to create symbols from strings:
```
>> Symbol("x") + Symbol("x")
2*x
>> {\[Eta], \[CapitalGamma]\[Beta], Z\[Infinity], \[Angle]XYZ, \[FilledSquare]r, i\[Ellipsis]j}
{\u03b7, \u0393\u03b2, Z\u221e, \u2220XYZ, \u25a0r, i\u2026j}
```
